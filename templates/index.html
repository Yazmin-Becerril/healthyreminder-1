Predicción de asistencia a citas médicas

Proyecto de Machine Learning orientado al sistema Healthy Reminder para identificar pacientes con riesgo de no asistir a su cita médica.

Objetivo

Detectar pacientes con mayor probabilidad de inasistencia para activar recordatorios y mejorar la gestión del sistema.

Mejor modelo

Logistic Regression con class_weight='balanced', priorizando recall sobre accuracy.

Recall final

0.8441

Enfoque

SQL + Pipeline + Feature Engineering + Manejo de desbalance + modelo guardado.

Resumen

El proyecto evolucionó desde un experimento académico básico hasta un flujo más cercano a producción, integrando almacenamiento en SQL, pipeline reproducible, comparación de modelos, manejo de desbalance y base de inferencia real.
